# ActionData - SetSpeed

等待交互动作参数

## 属性

| 名称        | 数据类型 | 说明                               |
| ----------- | -------- | ---------------------------------- |
| `speed` | number   | 要设置的速度 |

## 示例

```typescript
{
  "type": ActionType.SetSpeed,
  "data": {
    "speed": 0.4
  }
}
```
