# 获取任务详情

## 方法

## `getTaskDetailsById(taskId) -> {Promise.<any>}`

获取任务详情。



### 参数

| 名称   | 类型                          | 说明     |
| ------ | ----------------------------- | -------- |
| `taskId` | string | 任务标识 |



### 返回值 `Promise.<any>`
| 名称         | 类型                          | 说明     |
|------------| ----------------------------- | -------- |
| `taskInfo` | [TaskInfo](../../../Define/Define-TaskInfo) | 任务信息 |
